# API Reference

Complete API reference for NeuroLink.

## Core API

### Generate Text

```typescript
POST / api / generate;
```

### Stream Text

```typescript
POST / api / stream;
```

### Provider Status

```typescript
GET / api / status;
```

## MCP Integration

### List MCP Tools

```typescript
GET / api / mcp / tools;
```

### Execute MCP Tool

```typescript
POST / api / mcp / execute;
```

### MCP Server Status

```typescript
GET / api / mcp / status;
```

For complete API documentation, see [API Reference](../sdk/api-reference.md).
