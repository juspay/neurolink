ℹ️ RAG Information

How RAG (Retrieval-Augmented Generation) works:

1. Index: Documents are split into chunks and stored with metadata.

2. Query: Your question is used to find relevant chunks using keyword matching.

3. Context: Top matching chunks are provided as context to the AI.

4. Answer: AI generates an answer based on the retrieved context.

💡 Tip: Index multiple documents to build a comprehensive knowledge base. The more relevant documents you index, the better the answers!

ℹ️ How Context Summarization Works

1. Chat: Messages are stored with token counts for each session.

2. Monitor: Total tokens are tracked against the threshold.

3. Summarize: When threshold exceeded, older messages are summarized using the 9-section format.

4. Preserve: Recent messages (30%) are kept as-is for immediate context.

5. Merge: On subsequent summarizations, new content is merged with existing summary.

💡 9-Section Summary: Primary Intent, Technical Concepts, Files, Problem Solving, Pending Tasks, Task Evolution, Current Work, Next Step, Required Files
